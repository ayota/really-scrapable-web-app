Really Scrapable Web App is a tool to learn about web scraping

Super cool, right? There are lots of things to scrape on this website. Really Scrapable Web App (RSWA) is a nice, sandboxed environment where you can learn about the fine art of web scraping and data science. Fun right? You can install it locally on your computer, and it's totally open source. You can even fork it on Github and change it as you see fit!

So, what's so special about it? Firstly, it aims to test you and tax you, and make you learn. RSWA has a bunch of challenges of varying difficulty. They make you use technologies that are regularly used by data scientists.

If you have any suggestions for challenges or any comments, please Give us a shout! We'd love to hear from you

RSWA is in Alpha and is currently at version 0.0.1.
